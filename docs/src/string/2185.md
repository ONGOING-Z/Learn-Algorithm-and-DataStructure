## [Problem Info][problem_link]
给定一字符串数组和一字符串pref，查看数组中字符串前缀是pref的数量。

## Attention
- string.startsWith(String prefix)

## Answer - 1

- java

```java
class Solution {
    public int prefixCount(String[] words, String pref) {
        int res = 0;
        for (int i = 0; i < words.length; i++) {
            if (words[i].startsWith(pref)) {
                res++;
            }
        }
        return res;
    }
}
```
- time: 0ms
- beat 100%
- big O: O(n)
