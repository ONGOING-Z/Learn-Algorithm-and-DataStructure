## [Problem Info][problem_link]

给定一整数数组arr, 找出对于数组中每一个整数，找出小于它值的数字的个数。

```java
class Solution {
    public int[] smallerNumbersThanCurrent(int[] nums) {
        int[] res = new int[nums.length];

        for (int i = 0; i < nums.length; i++) {
            int base = nums[i];

            int temp = 0;
            for (int j = 0; j < nums.length; j++) {
                if (i == j) {
                    continue;
                }

                int num = nums[j];
                if (base > num) {
                    temp++;
                }
            }
            res[i] = temp;
        }

        return res;

    }
}
```
