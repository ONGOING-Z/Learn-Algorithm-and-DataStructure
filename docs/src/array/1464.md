## [Problem Info][problem_link]
给定一整数数组，找出数组中最大的两个值x/y，返回(x-1)*(y-1)

## Attention

## Answer - 1

- java

```java
class Solution {
    public int maxProduct(int[] nums) {
        Arrays.sort(nums);
        return (nums[nums.length - 1] - 1) * (nums[nums.length - 2] - 1);

    }
}
```
- time: 2ms
- beat 38.54%
- big O: O(n)
