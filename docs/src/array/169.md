## [Problem Info][problem_link]
找到数组中的众数。

## Attention

## Answer - 1
排序，找中位数。

- java
```java
class Solution {
    public int majorityElement(int[] nums) {
        Arrays.sort(nums);

        int n = nums.length;
        int mid = nums[n / 2];
        return mid;
    }
}
```

## Answer - 2
Boyer-Moore 投票算法
```java
class Solution {
    public int majorityElement(int[] nums) {
        int count = 0;
        int candidate = 0;

        for (int num: nums) {
            if (count == 0) {
                candidate = num;
            }
            count += (num == candidate) ? 1 : -1;
        }

        return candidate;
    }
}
```
