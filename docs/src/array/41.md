## [Problem Info][problem_link]
找到第一个缺失的正整数。

例如 1, 2, 3 缺失的就是4

1，3， 4 缺失的就是2.

## Answer - 1

- java
```java
class Solution {
    public int firstMissingPositive(int[] nums) {
        Arrays.sort(nums);
        int n = 1;
        for (int i = 0; i < nums.length; i++) {
            if (nums[i] <= 0) {
                continue;
            }
            if (nums[i] == n-1) {
                continue;
            }
            if (nums[i] != n) {
                return n;
            }
            n++;
        }
        return n;
    }
}
```
