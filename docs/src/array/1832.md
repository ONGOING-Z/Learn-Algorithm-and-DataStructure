## [Problem Info][problem_link]

查看一个字符串中是否出现了字母表中的所有字母。

## Attention

## Answer - 1

- java
```java
class Solution {
    public boolean checkIfPangram(String sentence) {
        int[] alphabet = new int[26];
        char[] chars = sentence.toCharArray();

        for (char ch: chars) {
            alphabet[ch - 'a']++;
        }

        for (int i = 0; i < alphabet.length; i++) {
            if (alphabet[i] == 0) {
                return false;
            }
        }
        return true;
    }
}
```
