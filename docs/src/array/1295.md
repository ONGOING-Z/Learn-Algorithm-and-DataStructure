## [Problem Info][problem_link]
给定int数组，找出数组中数字位数为偶数的个数。

## Attention

## Answer - 1

- java

```java
class Solution {
    public int findNumbers(int[] nums) {
        int res = 0;
        for (int i = 0; i < nums.length; i++) {
            String str = String.valueOf(nums[i]);
            if (str.length() % 2 == 0) res++;
        }
        return res;
    }
}
```
- time: 2ms
- beat 23.99%
- big O: O(n)
