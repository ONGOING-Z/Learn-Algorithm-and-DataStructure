## [Problem Info][problem_link]
给定一整数数组，将数组按非递减的方式进行排序，求出和原来数组相比，移动数字的次数。

## Attention
- int[] b = a.clone();
## Answer - 1
将数组先排序，然后和原来的数组相比，求出数字不一样的位置个数。

- java

```java
class Solution {
    public int heightChecker(int[] heights) {
        int[] newH = heights.clone();
        Arrays.sort(newH);
        int res = 0;
        for (int i = 0; i < heights.length; i++) {
            if (newH[i] != heights[i]) {
                res++;
            }
        }
        return res;
    }
}
```
- time: 1ms
- beat 64.81%
- big O: O(n)
