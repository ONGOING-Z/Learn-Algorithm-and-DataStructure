```java
class Solution {
    public int missingNumber(int[] nums) {
        Arrays.sort(nums);
        int start = 0;
        for (int i = 0; i < nums.length; i++) {
            if (start == nums[i]) {
                start++;
                continue;
            }
        }

        return start;
    }
}
```
优化：不需要start临时变量，直接使用循环里的i
```java
class Solution {
    public int missingNumber(int[] nums) {
        Arrays.sort(nums);
        for (int i = 0; i < nums.length; i++) {
            if (i == nums[i]) {
                continue;
            }
            return i;
        }

        return nums.length;
    }
}
```
